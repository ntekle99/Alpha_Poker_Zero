# Alpha Poker Zero - Frontend

React-based poker table UI matching PokerNow's visual design.

## Setup

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Features

- **Exact PokerNow Visual Design**: Green oval poker table with all visual elements
- **Player Positions**: Left and right player seats with cards and chip stacks
- **Action Buttons**: CALL, RAISE, CHECK, FOLD buttons
- **Game State Display**: Pot, blinds, dealer button, community cards
- **Chat/Log Panel**: Message display and input
- **Responsive Design**: Adapts to different screen sizes

## Components

- `PokerTable.jsx` - Main poker table component
- `App.jsx` - Root application component
